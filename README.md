# Project 1: NYC Taxi Ridership Analysis (Automatidata)
## Google Advanced Data Analytics Professional Certificate

### 🚕 Project Overview
This project involves working with the New York City Taxi and Limousine Commission (TLC) as a data consultant for the fictional firm **Automatidata**. The goal is to prepare and analyze taxi ridership data to help optimize operations and provide data-driven insights for future predictive modeling.

### 📊 Key Files
- `Activity_Course 2 Automatidata project lab.pdf`: Data inspection, cleaning, and identification of key variables.
- `Activity_Course 3 Automatidata project lab.pdf`: Exploratory Data Analysis (EDA) and visualization using Matplotlib and Seaborn.

### 🛠 Skills & Tools
- **Data Cleaning:** Handling inconsistent formats and preparing dataframes for analysis.
- **Exploratory Data Analysis (EDA):** Identifying outliers in trip distance and duration.
- **Visualization:** Box plots for outlier detection and time-series plots for ridership trends.
- **Tools:** Python (Pandas, Numpy, Seaborn, Matplotlib), Tableau.

### 💡 Key Insights
- Identified specific outliers in the `trip_duration` and `fare_amount` columns that require treatment before modeling.
- Discovered seasonal trends in ridership that can help the TLC better manage driver supply during peak months.
